We use the collected information to:

Data Storage

All image processing is done locally on your device. We do not store or transmit your images to any servers.

Third-Party Services

We use the following third-party services:

Updates

The Windows version includes automatic updates to provide security fixes and new features. You will be notified before any update is installed.
